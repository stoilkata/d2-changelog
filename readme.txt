general:

Implemented Terror Zones - every hour, on the hour, 5 random zones will be terrorized.
Density will be increased 3 times on all difficulties
On Normal and Nightmare difficulty, monster lvl will be increased by 10%.
On Hell, monster lvl will be set to 90.

The server will announce the terrorized zones 1 minute past the hour. Additionally you can use the "/tz" command to receive a notification about the zones.
